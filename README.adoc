= Image Indexer Core

== Summary
Core library for Image Search Indexing logic, including ONNX runtime bindings for CLIP-based image embedding and search.

== Architecture
- **Inference**: Uses `dgml` (Pure D) for CLIP model inference.
- **Image Processing**: Validates and preprocesses images using `imageformats`.
- **Indexing**: Uses `d-vector-search` for storage and retrieval.


== Usage
See `source/app.d` for example.

== Requirements
- **DGML**: Pure D tensor library (see `../dgml`).
- **D-Vector-Search**: Pure D vector store (see `../d-vector-search`).
